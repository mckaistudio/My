# VoicePace AI Extension

This repository contains the complete VoicePace AI extension including all necessary HTML pages, JavaScript files, and utilities.

## Files Structure
- manifest.json
- background.js
- content.js
- hack.js
- popup.html
- popup.js
- settings.html
- settings.js
- dashboard.html
- dashboard.js
- media-manager.html
- media-manager.js
- utils/
  - storage.js
  - logger.js
  - validator.js
- styles/
  - common.css
  - themes.css

## Installation
1. Clone this repository.
2. Load the extension in your browser's extension settings.

## Usage
- Open the extension to access features.
